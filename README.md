# SolarNode
Weatherproof enclosure design for RAK4631
# Background
This build was loosely inspired by the following [reddit post](https://www.reddit.com/r/meshcore/comments/1su4cal/overbuilt_dual_mesh_node/). Much thanks to u/Database121 for the inspiration and solid parts list to work from.
With that being said, the overall design philosophy is to build a mesh node that is both as rugged as possible while also being as simple as possible and relatively affordable to build.
